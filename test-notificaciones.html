Instrucciones de Configuración

    
En tu proyecto Supabase, en el SQL editor, ejecuta:

    
select set_config('app.settings.process_outbox_url','https://<PROJECT-REF>.functions.supabase.co/process-outbox', true);
select set_config('app.settings.service_role_token','<SERVICE_ROLE_KEY>', true);
-- Luego invoca manualmente:
select public.invoke_process_outbox();
    

    
Y asegura en Secrets de Edge Functions:

    
    
      
  • VAPID_PUBLIC_KEY: clave pública (Base64 URL-safe, formato P-256)
    • 
      
  • VAPID_PRIVATE_KEY: clave privada correspondiente
    • 
      
  • VAPID_SUBJECT: mailto o URL
    • 
      
  • SUPABASE_SERVICE_ROLE_KEY: service role para autorización Bearer
    • 
      
  • SUPABASE_URL: URL de tu proyecto
    •
